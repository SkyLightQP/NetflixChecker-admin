# NetflixChecker-admin ![Next JS](https://img.shields.io/badge/Next-black?style=for-the-badge&logo=next.js&logoColor=white)

> NetflixChecker 관리자 대시보드

## 시작하기
```bash
pnpm install

pnpm build

pnpm start
```

## 환경변수
```dotenv
# NetflixChecker API URL
NEXT_PUBLIC_API_URL=http://localhost:3000
```
